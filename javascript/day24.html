map method will not affect the original array

in map method this is correct syntax map(function(element,index)) we cannot write map(function(index,element))

higher order functions of array

  • javascript array is having 3 very important higher order fucntions they are
    1. map
    2. filter
    3. reduce

map

map mehtod is used to perform common operation on each element of the array

it will return a new array without madifying the original array

syntax:

array.map((element,index) =>{ return statement})

it takes a callback function which will exicute for each and every element of the array

filter

it is used to filter the elements in an array based on the condition

it returns a new array without modifying original array

it takes a call back fucntion as the argument , that fucntion exicutes for each and every element of the array

syntax:array.filter((element,index) =>{ return statement})

reduce

it is used to reduce the array into single value

it returns single value as the output

it takes a callback function which will exicute for each and every element of the array

syntax:array.reduce((current,next) =>{return statement})
