hoisting

hoisting is the default behaviour of javascript to move declarations to the top of the scope

hoisting is of two types

  1. variable hoisting
  2. function hoisting

varible hoisting

it is the behaviour of the javascript to move variable declaration to the top of the scope at the runtime

because of this in javascript a variable can be used even it is declared

variable hoisting is only possible with var keyword

types of fucntions in javascript there are 7 types

as varible hoisting causes lot of problems in order to avoid it we will use const and let

for const variable we have to declare and intialize at the same time so hoisting is not possible

function hoisting

it is the behaviour of the javascript to move function declaration to the top of the code at the runtime

because of this in javascript a function can be called even before we declare it

function hoisting is possible in general or normal functions

function hoisting is possible with only normal funcation

as function hoisting causes lot of problems in order to overcome it we will use anonymus function and arrow fucntions
